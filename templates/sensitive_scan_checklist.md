# Sensitive Scan Checklist

提交、打包、迁移或开 PR 前使用本清单。这里不保存任何真实密钥、账号或媒体文件。

## 禁止上传

- [ ] 没有环境配置文件
- [ ] 没有钉钉机器人地址
- [ ] 没有钉钉签名密钥
- [ ] 没有 API key、access token、密码或真实凭据
- [ ] 没有即梦账号、Cookie、浏览器 session 或登录状态
- [ ] 没有原视频
- [ ] 没有 AI 生成视频
- [ ] 没有音频文件
- [ ] 没有截图
- [ ] 没有抽帧图片
- [ ] 没有 Logo 原文件

## 允许上传

- [ ] Markdown 流程文档
- [ ] Markdown 检查模板
- [ ] 脱敏 JSON 状态模板
- [ ] 小型文本报告
- [ ] 脚本源码

## 迁移到正式仓库前

- [ ] 重新运行正式仓库要求的敏感信息扫描命令
- [ ] 如果扫描出现任何风险，停止提交并请负责人确认
- [ ] 只迁移已经确认安全的文档、模板、脚本和小型文本报告

## 检查结论

- [ ] 通过，可以提交或迁移
- [ ] 不通过，需要删除或脱敏后重新检查
