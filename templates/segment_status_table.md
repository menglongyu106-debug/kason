## 状态词建议

| 状态 | 含义 |
|---|---|
| 未开始 | 还没有处理 |
| 检查中 | 正在进行 preflight 或人工核对 |
| 已通过 | 已完成且可进入下一步 |
| 阻塞 | 有问题，必须先修正 |
| 跳过 | 本轮不处理，需要写明原因 |

## 更新规则

- 不要把正式生产 run 的状态写进本表。
- 每个 Segment 进入生成前，必须先有通过的 preflight。
- 没有人工确认，不得把生成状态改成已通过。
- 如果出现字幕、Logo、对话错误或提示词错误，状态应标为阻塞。
